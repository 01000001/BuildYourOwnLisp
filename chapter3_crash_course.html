Program Structure

C programs are structured very simply. They only consist only of function definitions and structure definitions. This means a C source file is simply a list of functions and data types. These functions can call each other or themselves, and use the data types declared.

It is also possible to call other libraries of functions, or use their data types.

As we saw in the previous chapter, the actual execution of a C program starts in the function called main and calls other functions to do more complicated processing.

Variables & Types

Every variable in C has an explicit type. As well as being able to define our own data types, some native types are built into C. Here is a few of them with some basic descriptions.

void Empty Type
char Single Character/Byte
int Integer
long Integer that can hold larger values
float Decimal Number
doubleDecimal Number with more precision

We declare a new variable by writting the type name then the variable name, and optionally setting it to some value. For example to create a new int called count we could write the following inside a function.

int count;

or to declare it and set the value

int count = 10;

Functions

To declare a function in C, one writes the name of the type it returns, the name of the function, and then in parenthesis a list of arguments/inputs to this function separated by commas. Each argument is a type name and a variable name.

The contents of the function is put inside curly brackets {}, and lists all of the statements the function executes, terminated by semicolons ;. A return statement is used to let the function end and output a value.

For example a function that takes two int variables called x and y and adds them together could look like this.

int add_together(int x, int y) {
  int result = x + y;
  return result;
}

Structs

To declare our own types we can use the struct keyword in conjunction with the typedef keyword.

Structs are like several C variables bundled together in a single package. We could create a struct called point that represents a point in 2D space by packaging together two float (decimal) values called x and y. It would look like this.

typedef struct {
  float x;
  float y;
} point;

We should place this at the top of our source file so it becomes avaliable to use as a type for our functions. As a type it is no different to the native types and so we can use it in all the same ways. We can access each individual field using a dot ., followed by the field name such as x.

point p;
p.x = 0.1;
p.y = 10.0;

Pointers

A pointer is a variation on a normal type where the type name is suffixed with a asterix *, for example int*. These are a difficult part of C and will be explained in much greater detail in later chapters. For now it is good to simply know they exist, and how to spot them. They can be used in a number of different ways such as for strings and lists.

We'll see all of these uses in later chapters, where I'll explain everything greater depth. For now just don't let them scare you off.

Strings

Strings in C use the pointer type char*. Under the hood they are a list of bytes terminated by a special byte called the null terminator. Strings are a complicated, but integral and important part of C, which we'll learn to use effectively.

They can also be declared literally by putting text between quotation " marks, for example "Hello, World!".

Just remember that if you see char*, you should read it as string.

Conditionals

To perform code under some condition we can use the if statement. It is written as if followed by some condition in parenthesis (), followed by the code to execute in curly brackets {}. An if statement can be followed by an optional else followed by more statements in curly brackets {}. This code will be performed in the case the conditional statement is not true.

For example if we wish to check if an int called x is greater than 10 or not we might write the following.

if (x > 10) {
  puts("x is 0!");
} else {
  puts("x is not 0!");
}

Inside a conditional any value that is not 0 will evaluate to true. This is important to remember as many conditions use this to check something implicitly.

Loops

There are two main loops in C.

The first is a while loop. This loop repeatedly executes a block of code until some condition becomes false. It is written as while followed by some condition in parenthesis (), followed by the code to execute in curly brackets {}. For example a loop that counts downward from 10 to 1 could be written as follows.

int i = 10;
while (i > 0) {
  puts("Loop Iteration");
  i = i - 1;
}

The secon kind of loop in C is a for loop. Rather than a condition this loop contains three statements separated by semicolons ;. These are an initialiser, a condition and an incrementer. The initaliser is performed before the loop starts, the condition is checked at the end of each iteration of the loop, and if false the loop is exited. The incrementer is performed before the next iteration of the loop.

For example to write a loop that counts up from 0 to 9 we might write the following. In this case the ++ operator increments the variable i.

for (int i = 0; i < 10; i++) {
  puts("Loop Iteration");
}

Notice how this is more compact than the while loop for counting. This is often what it is used for.
